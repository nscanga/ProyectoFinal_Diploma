Crear Cliente

?? Propósito: Registrar nuevos clientes en el sistema.

Campos del Formulario

Campo Descripción Obligatorio
Nombre Nombre completo o razón social del cliente ? Sí
Dirección Dirección física del cliente ? No
Email Correo electrónico de contacto (debe contener @) ? Sí
Teléfono Número de teléfono (solo números) ? No
CUIT Número de CUIT/CUIL del cliente ? No
Activo Indica si el cliente está activo en el sistema ? Sí

Pasos para Crear un Cliente

1. Complete todos los campos obligatorios.
2. Verifique que el email tenga formato válido (ejemplo: cliente@empresa.com).
3. Si el teléfono contiene un @ por error, el sistema le avisará.
4. Haga clic en "Guardar" para crear el cliente.

Errores Comunes

  • Email sin @: El sistema requiere un formato válido de email.
  • Campo vacío: Los campos marcados como obligatorios deben completarse.
  • Email en campo teléfono: El campo teléfono solo acepta números.
