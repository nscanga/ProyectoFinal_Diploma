Stock list

Description

Displays the current availability for each product, including category, stock unit type, unit price, and status. The information comes from StockService.ObtenerStockConDetalles() and is presented in an automatically formatted grid.

How to access

  1. On the main screen select Stock â†’ Show stock.
  2. The grid loads and is formatted when the form opens.

Main columns

Column Description
Product Name of the associated product.
Category Product family (Fresh chicken, Eggs, etc.).
Quantity Available stock in the defined unit.
Stock type Unit of measure (Unit, Box, Kilogram, etc.).
Price Unit price formatted as currency.
Active Indicates whether the product is enabled for sale.

Available actions

  • Edit stock: Select a row and press the corresponding button to adjust quantities.
  • Contextual help: Press F1 to open this guide at any time.

Tips

Sort the Quantity column to identify low-stock products.
Combine this view with the low stock report to anticipate replenishments.
