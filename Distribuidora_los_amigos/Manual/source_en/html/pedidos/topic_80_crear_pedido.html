Register a new order

Description

Use this screen to create sales orders for registered customers. Select the customer, initial status, and the products that will be part of the order.

How to access

  1. From the main screen choose Orders ‚Üí Create order.
  2. The CrearPedidoForm form opens with the customer and product lists loaded.

Main controls

Control Description Required
Customer (comboBoxSeleccionCliente) Customer requesting the order. Yes
Status (comboBoxEstadoPedido) Initial order status (Pending, In preparation, etc.). Yes
Date (dateTimePickerCrearPedido) Order creation date. Filled automatically with the current date. Yes
Product list (dataGridViewProductos) Catalog of available products with price, category, and dates. Yes (to add items)
Quantity (numericUpDown1) Quantity to add for the selected product. Yes
Order details (dataGridViewDetallePedido) Summary of added products with quantities and subtotals. Yes
Save order Button that registers the order and deducts stock. ‚Äî

Step by step

  1. Select the customer and the initial status.
  2. Choose a product in the available product grid.
  3. Enter the quantity and press Add product. The system validates available stock.
  4. Repeat the process for each product that must be included in the order.
  5. Review the order details and the total calculated automatically.
  6. Press Save order. The order is created with its details and the corresponding stock is deducted.

Validations

  • A customer must be selected and at least one product must be added.
  • It is not possible to add quantities greater than the available stock.
  • If the same product is added again, the quantity is accumulated as long as stock allows it.
  • The order status must be selected; it cannot remain blank.

Tips

Use the DataGridView search (Ctrl+F) to find products quickly.
Keep stock quantities up to date to avoid rejections due to lack of inventory.
