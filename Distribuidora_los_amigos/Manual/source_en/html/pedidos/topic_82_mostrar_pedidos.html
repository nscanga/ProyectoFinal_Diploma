View Orders

Introduction

The order list centralises every sale registered in the system, showing customer name, current status, total amount and number of items. From here you can create new orders, edit their status, delete them or open the detailed view of each one.

It is the main dashboard for tracking the fulfilment workflow and quickly identifying orders that require attention.

How to Access

To access this feature:

  1. From the main menu choose Orders ‚Üí View orders.
  2. The MostrarPedidosForm loads with the grid populated.

Procedure

1 Select the order

Sort the grid or use the scrollbar to locate the order you want to manage, then click the row.

2 Change the status quickly

Choose the new status from the combo on the lower-left corner and click Change Status. The system asks for confirmation before saving.

3 Execute additional actions

Use the action buttons to create, edit, delete or inspect details. Each button validates that a row is selected and displays informative messages when necessary.

4 Refresh the information

Click üîÑ Refresh after any modification to update the grid and the order counter.

System Validations

The form performs the following checks:

  • Shows a warning if you try to edit, delete or view details without selecting a row.
  • Requires a status selection before allowing the quick change and confirms the action with the user.
  • When deleting, calls PedidoService.EliminarPedido which restores stock and removes order details before deleting the order.
  • If no orders are found, an informational message is displayed.

Tips and Recommendations
