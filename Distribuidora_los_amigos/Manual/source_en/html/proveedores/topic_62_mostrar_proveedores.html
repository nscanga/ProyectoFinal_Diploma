Supplier list

Description

Displays all registered suppliers and allows you to create, edit, and delete records. Ideal for keeping the list of business partners up to date.

How to access

  1. From the main screen select Suppliers ‚Üí View suppliers.
  2. The grid loads automatically with the stored data.

Available actions

  • Create supplier: opens the creation form.
  • Edit supplier: requires selecting a row and opens the edit form.
  • Delete supplier: asks for confirmation and removes the record.

Validations

  • Select a row in the grid before editing or deleting.
  • Deletion requires confirmation and displays localized messages.
  • Errors are captured, shown on screen, and logged.

Tips

Use the grid filters (such as sorting by name) to quickly locate a supplier.
After making changes press the refresh button to update the view.
