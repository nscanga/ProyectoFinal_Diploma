Update supplier information

Description

Allows you to modify existing supplier information such as contact details, category, and active status.

How to access

  1. Open Suppliers ‚Üí View suppliers.
  2. Select the supplier and press Edit.

Editable fields

Field Description Required
Name Name or business name. Yes
Address Main address. Yes
Email Valid contact email. Yes
Phone Primary phone number. Yes
Category Classification within the supply chain. Yes
Active Lets you enable or disable the supplier. Yes

Step by step

  1. Update the necessary fields.
  2. Press Save. The system validates the data through ProveedorService.ModificarProveedor.
  3. The message ‚ÄúSupplier updated successfully‚Äù appears and the form closes.

Validations

  • Same checks as during creation: required fields and valid email address.
  • Messages are shown in the language currently selected.

Tips

Use this screen to temporarily deactivate suppliers by toggling the Active checkbox.
After saving, refresh the list to confirm the changes.
