Low Stock Report

Introduction

The low stock report lists every active product whose inventory is below the configured threshold. It highlights critical items so you can prioritize replenishment before daily operations are affected.

Use this view during warehouse control routines or to justify urgent purchasing decisions. The information is calculated in real time from the stock module and classifies each product by its severity level.

How to Access

To open this report:

  1. From the main menu, go to Reports.
  2. Select Low Stock.
  3. The Low Stock Report window opens with the latest data.

Control Description

Each control available on the screen is described below:

Procedure

1 Define the threshold

Enter the desired minimum value in Low Stock Threshold and click Update to recalculate the report.

2 Review the indicators

Check totals and the number of critical, very low, and low products to understand the situation.

3 Prioritize actions

Sort or filter the grid by Status or Quantity to decide which items to restock first.

4 Export information

If you need to share the analysis, click Export to Excel and save the generated CSV file.

System validations

The system enforces the following rules:

  • The threshold must be an integer greater than zero; otherwise you are asked to adjust the value.
  • Only active products with current stock less than or equal to the configured threshold are displayed.
  • Export is blocked when the grid is empty to avoid creating blank files.

Tips and recommendations

Adjust the threshold according to product turnover. For fast-moving items, use higher values to anticipate stock-outs.
Combine this report with the purchasing history to plan replenishment orders based on real data.
