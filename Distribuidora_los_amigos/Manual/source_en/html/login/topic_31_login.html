System Login

Introduction

Login is the first step to access the Distribuidora Los Amigos Management System. This process verifies your identity and loads the permissions associated with your user.

ℹ Important: Each user has unique credentials that should not be shared with others. The system records all actions associated with your user.

How to Login

1 Launch the Application

Open the "Distribuidora Los Amigos" application from the desktop shortcut or from the Windows start menu.

2 Login Screen

When launching the application, the login window will automatically appear.

3 Enter Credentials

Complete the following fields:

  • Username: Your username (no spaces, case sensitive)
  • Password: Your personal password (hidden for security)
4 Select Language (Optional)

You can select the interface language before logging in. Available options:

  • 🇪🇸 Spanish
  • 🇺🇸 English
  • 🇵🇹 Portuguese
5 Login

Click the "Login" button or press Enter.

Field Description

Field Description Format Required
Username Username assigned by the administrator Alphanumeric text (case sensitive) Yes
Password User's personal password Text (hidden characters for security) Yes
Language System interface language Dropdown list No (default: last used)

System Validations

During login, the system verifies:

  • ✓ That the user exists in the system
  • ✓ That the password is correct
  • ✓ That the user is active (not blocked)
  • ✓ That there is no other session open with the same user
  • ✓ User data integrity (DVH)

First System Access

⚠ Mandatory Password Change: If it's your first access or the administrator reset your password, the system will require you to change it immediately for security reasons.

New Password Requirements

  • Minimum 8 characters
  • At least one uppercase letter
  • At least one lowercase letter
  • At least one number
  • At least one special character (!@#$%^&*)
  • Cannot be the same as the last 3 passwords

Common Errors and Solutions

Error: "Incorrect username or password"

Cause: The entered credentials are not valid.
✓ Solutions:
  1. Verify that the username is spelled correctly (case sensitive)
  2. Make sure Caps Lock is off
  3. Verify that the password is correct
  4. If you forgot your password, use the "Forgot your password?" option

Error: "User is blocked"

Cause: The user was blocked due to multiple failed attempts or by the administrator.
✓ Solution:

Contact the system administrator to unlock your account.

Error: "There is already an active session for this user"

Cause: The system detected an already open session with your credentials.
✓ Solutions:
  1. Check if you have the system open on another computer
  2. If the application closed unexpectedly, wait 5 minutes and try again
  3. Contact the administrator to close the previous session

Error: "Database connection error"

Cause: The system cannot connect to the database server.
✓ Solutions:
  1. Verify your network connection
  2. Verify that the database server is online
  3. Contact the system administrator

Error: "Data tampering detected"

Cause: The system detected data inconsistencies (DVH failure).
✓ Solution:

URGENT: Contact the administrator immediately. This error indicates possible data corruption or tampering.

Security Recommendations

Tips to protect your account:
  • 🔒 Never share your password with anyone
  • 🔄 Change your password periodically (every 90 days recommended)
  • 🔑 Use unique passwords - do not reuse passwords from other systems
  • 📝 Do not write down your password in visible places
  • 🚪 Logout when not using the system
  • 👀 Do not leave the system unattended with your session open
  • 🖥️ Lock your computer if you step away from your desk
⚠ Lockout Policy: After 5 failed login attempts, your account will be automatically locked for 30 minutes as a security measure.

Password Recovery

If you forgot your password:

  1. Click the "Forgot your password?" link on the login screen
  2. Enter your username and registered email
  3. Answer the configured security questions
  4. You will receive a recovery code by email
  5. Enter the code and set a new password
ℹ Note: The recovery code is valid for 15 minutes. If it expires, you must request a new one.

After Logging In

Once you successfully login:

  • The system Main Screen will open
  • The system will load your preferred language
  • Only the options you have permission for will be enabled
  • Your username will appear in the status bar
  • The login will be recorded in the system event log
