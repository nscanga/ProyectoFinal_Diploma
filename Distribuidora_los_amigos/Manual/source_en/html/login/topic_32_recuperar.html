Password recovery

When should you use this screen?

Use this functionality when a user forgets their password or needs to reset it for security reasons. The system sends a code (token) to the registered email address and allows you to set a new password safely.

How to access

  1. On the sign-in screen click Forgot your password?
  2. The Password recovery modal window opens.
  3. Complete the indicated steps and close the window when finished.

Form structure

Section Description
Token request Allows you to enter the username and send the recovery code by email.
Token entry Panel enabled after sending the email. It asks for the token, new password, and confirmation.
System messages Information label that displays the result of the operation (success or error).

Step by step

1 Enter username

Type the username exactly as it was registered and press Send recovery code.

2 Token reception

The system generates a unique token and sends it to the associated email. The button is disabled while processing.

3 Complete the change panel

Enter the received token, the new password, and its confirmation. Then press Change password.

4 Finish

If the data is valid a success message appears and the form closes. You can sign in with the new password.

Validations and messages

  • The username is required to send the token.
  • The token cannot be empty and must match the one sent by email.
  • The new password must be at least 6 characters long and match its confirmation.
  • All messages are automatically translated according to the selected language.

Required permissions

No specific patent is required. Any user with access to the login can request a token if their email address is registered in the system.
