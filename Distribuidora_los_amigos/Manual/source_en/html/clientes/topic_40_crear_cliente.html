Create Customer

Introduction

This functionality allows you to register new customers in the system. Customers are the people or companies to whom product sales will be made.

A properly registered customer will allow you to manage their orders, check their purchase history, and generate specific reports.

How to Access

To access this functionality:

  1. From the main menu, go to Customers
  2. Select Create Customer
  3. The Create Customer window will open

Field Description

Each field in the form is described below:

Field Description Type Required
Name Full name of the customer or company business name Text (max. 100 characters) Yes
Last Name Customer's last name (if individual) Text (max. 100 characters) No
DNI/CUIT National Identity Document or Tax ID of the customer Numeric (8-11 digits) Yes
Email Customer's email address Valid email No
Phone Contact phone number Numeric/Text No
Address Complete address of the customer Text (max. 200 characters) No
Postal Code Postal code of the address Numeric (4 digits) No
Active Indicates if the customer is active in the system Checkbox (Yes/No) Yes (default: Yes)

Procedure to Create a Customer

1 Enter Basic Information

Complete the Name and DNI/CUIT fields, which are required. If it's an individual, also complete the Last Name.

2 Add Contact Information

Enter the customer's Email and Phone. Although optional, it's recommended to complete them to facilitate communication.

3 Complete Address

Enter the complete Address and Postal Code. This is important for delivery management.

4 Verify Status

Make sure the Active checkbox is checked if you want the customer to be available for placing orders immediately.

5 Save the Customer

Click the Save button or press Ctrl + S. If all data is valid, the customer will be created successfully.

System Validations

The system performs the following validations:

  • The Name and DNI/CUIT fields cannot be left empty
  • The DNI/CUIT must be unique - there cannot be two customers with the same document
  • If an Email is entered, it must have a valid format (example@domain.com)
  • The Name cannot exceed 100 characters
  • The DNI/CUIT must contain only numbers
  • The Postal Code must be a 4-digit numeric value

Tips and Recommendations

Complete all possible fields, even if they are not required. This will facilitate communication with the customer and order management in the future.
Double-check the DNI/CUIT before saving, as this data is unique and cannot be duplicated. If you try to create a customer with an existing DNI/CUIT, the system will display an error.

After Creating the Customer

Once the customer is successfully created:

  • The customer will appear in the Show Customers list
  • You can assign orders to them from Create Order
  • You can modify their data from Modify Customer
  • You can check their purchase history in the reports
