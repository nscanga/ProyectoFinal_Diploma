View Customers

Introduction

The customer list displays every registered customer and provides shortcuts to create, edit or remove records. The grid shows all data returned by ClienteService.ObtenerTodosClientes() and is automatically refreshed after each operation.

Use this screen to keep track of your customer base, search for contact information and maintain data quality.

How to Access

To access this feature:

  1. From the main menu choose Customers ‚Üí View customers.
  2. The MostrarClientesForm opens with the grid already populated.

Screen Elements

Element Description Required
Customer grid (dataGridView1) Shows every customer attribute. Rows can be sorted and must be selected before launching an action. Yes
Create Opens the Create Customer form. When the window closes, the grid refreshes automatically. No
Modify Requires a selected row and opens Modify Customer with the chosen record. No
Delete Requests confirmation and removes the customer through ClienteService.EliminarCliente. No
Refresh Reloads the grid by calling CargarClientes(). Useful if changes were made from another window. No

Procedure

1 Search and select

Use column sorting or the scrollbar to find the customer you need. Click the row to highlight it.

2 Launch the desired action

Choose Create, Modify or Delete. The form validates the selection and shows meaningful feedback if none is provided.

3 Refresh the list

After saving or deleting, click Refresh to ensure the grid reflects the latest data.

System Validations

The form performs the following checks:

  • Shows a warning if you try to modify or delete without selecting a customer.
  • Asks for confirmation before deleting a customer.
  • Automatically reloads the grid when a child form closes to avoid stale data.
