Remove a product from the catalog

Description

Permanently deletes a product and its associated stock. Use this option when an item will no longer be sold.

How to access

  1. Open Products ‚Üí View products.
  2. Select the product to remove and press the Delete button.

Step by step

  1. Select the row for the product you want to delete.
  2. Press Delete. The system displays a confirmation dialog.
  3. Confirm the operation. ProductoService.EliminarProducto runs, removing the product and its stock.
  4. The grid refreshes automatically to reflect the changes.

Validations and messages

  • If no row is selected the message ‚ÄúSelect a product to delete‚Äù appears.
  • The confirmation dialog warns that the deletion is irreversible.
  • Errors are displayed and logged for follow-up.

Recommendations

Before deleting, check that there are no pending orders using the product.
If you only want to stop selling it, consider disabling it from the stock module instead of deleting it.
