Product list

Description

Displays the product catalog and lets you perform create, edit, delete, and refresh actions. Products appear in a simple grid that can be sorted by column.

How to access

  1. From the main screen select Products ‚Üí View products.
  2. The list loads automatically by calling ProductoService.ObtenerTodosProductos().

Available actions

  • Create product: opens the creation form.
  • Edit product: requires selecting a product in the grid.
  • Delete product: asks for confirmation and removes the product and its associated stock.
  • Refresh: queries the database again to reflect recent changes.

Validations

  • You must select a row before editing or deleting; otherwise the system shows a message.
  • Deletion requires confirmation. If accepted, related stock records are removed as well.
  • Errors are displayed on screen and logged.

Tips

Sort the Name column to group similar products.
Combine this view with the stock report to identify products with low availability.

Required permissions

The VER_PRODUCTOS, AGREGAR, MODIFICAR, and ELIMINAR patents are required depending on the action.
