Register a new product

Description

Adds products to the catalog. Each product is recorded with its category, price, relevant dates, and initial stock. Creating the product automatically generates an associated stock record.

How to access

  1. On the main screen select Products ‚Üí Add.
  2. The CrearProductoForm form opens centered on the screen.

Form fields

Field Description Type Required
Name Commercial name of the product. Text Yes
Category Select from predefined categories (Fresh chicken, Eggs, etc.). ComboBox Yes
Price Unit selling price. Must be greater than zero. Numeric Yes
Entry date Date when the product enters inventory. DatePicker Yes
Expiration date Optional. Enable the DateTimePicker checkbox to register it. DatePicker No
Initial quantity Opening stock that will be created in the warehouse. Numeric Yes
Stock type Unit, kilogram, box, tray, etc. Defines the unit of measure. ComboBox Yes

Step by step

  1. Fill in the name and choose the appropriate category.
  2. Enter the unit price (e.g., 1200.50). The system validates that it is positive.
  3. Set the entry date and, if needed, enable and set the expiration date.
  4. Specify the initial quantity and the stock type that will be created.
  5. Press Save. The product and its opening stock are registered.
  6. The message ‚ÄúProduct created successfully‚Äù appears and the form remains ready to add another item.

Validations

  • The price must be numeric and greater than zero.
  • A category and stock type must be selected.
  • If an expiration date is entered, it cannot be earlier than the entry date.
  • Name and category cannot be left blank.

Recommendations

Use consistent categories to simplify product-type reports.
If the product has no expiration, clear the control to avoid incorrect dates.
