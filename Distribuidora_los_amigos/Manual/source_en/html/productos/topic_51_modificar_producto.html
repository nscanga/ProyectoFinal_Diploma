Update an existing product

Description

Allows you to edit information for registered products. Use it to correct prices, update expiration dates, or change the assigned category.

How to access

  1. Open Products ‚Üí View products.
  2. Select a product and press Edit. This form opens with the data preloaded.

Editable fields

Field Description Required
Name Product name. Yes
Category Drop-down list with the categories defined in the system. Yes
Price Must be numeric and greater than zero. Yes
Entry date Can be adjusted if needed. Yes
Expiration date Optional. Check the control if the product has an expiration date. No

Step by step

  1. Review the existing data and apply the necessary updates.
  2. Press Save. The form validates price, category, and dates.
  3. If everything is correct the message ‚ÄúProduct updated successfully‚Äù appears and the form closes.

Validations

  • Price must be a positive number.
  • Category cannot be left blank.
  • If an expiration date exists, it must be the same as or later than the entry date.

Tips

Use this screen to register expiration dates when new stock arrives.
After editing, click ‚ÄúRefresh‚Äù on the product list to view the changes.
