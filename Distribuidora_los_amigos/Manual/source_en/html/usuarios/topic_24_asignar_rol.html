Assign roles (families) to users

Overview

This screen links a user with a previously configured family (role). When the family is assigned the user inherits all included patents and their menu updates immediately.

How to access

  1. From the main screen select Management â†’ Assign role.
  2. The form displays two drop-down lists: users and families.

Available controls

Control Description Type Required
comboBoxUsers List of users retrieved by UserService.GetAllUsuarios(). ComboBox Yes
comboBoxRoles Available families (roles) loaded with FamiliaService.GetAllFamilias(). ComboBox Yes
Assign Button that runs FamiliaService.AsignarFamiliaAUsuario and logs the action. Button â€”

Step by step

  1. Select the user in the first combo. The name is recorded in the log for auditing.
  2. Choose the family to assign. It can include patents and subfamilies.
  3. Press Assign. If everything is correct a success message appears.
  4. The user can now access the features enabled by the selected family.

Validations and messages

  • Selecting both a user and a family is mandatory. Otherwise a translated message is shown.
  • Successful operations are logged with the user performing the assignment.
  • Database errors are reported to the user and recorded for analysis.
