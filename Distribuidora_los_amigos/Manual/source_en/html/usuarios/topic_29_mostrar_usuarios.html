User and role list

Description

This screen displays the system users along with their assigned family and status (active or disabled). It uses the Syncfusion SfDataGrid, allowing you to sort and filter information quickly.

How to access

  1. From the main screen select Management â†’ View users.
  2. The grid populates automatically using UserService.GetUsuariosConFamilasYPatentes().

Grid columns

Column Description Filterable
User Username. Yes
Family Role assigned to the user (groups patents). Yes
Status Active / Disabled. Yes

Available actions

  • Sort columns by clicking the header.
  • Filter by text using the gridâ€™s filter row.
  • Refresh translations by pressing F1 to open the help or by switching the application language.

Security and auditing

  • Data loading is logged with the number of users displayed.
  • Database errors are handled with ErrorHandler and localized messages are shown.
  • The form unsubscribes from the language service when closed to free resources.

Tips

Use the family filter to quickly verify that each role has the expected users.
Combine this view with the audit report to detect recent modifications.
