Create permission roles (families)

Description

Families group permissions (patents) to simplify assigning features to users. This screen lets you create new families by selecting the patents that will belong to the role.

How to access

  1. Open the menu Management ‚Üí Create role.
  2. The CreateRolForm form opens with the list of available patents.

Form controls

Step by step

  1. Enter the name of the new family.
  2. Select the patents that will be part of the role (multiple selections allowed).
  3. Press Create family.
  4. The system displays a success message and logs the operation.

Validations

  • At least one patent must be selected; otherwise a warning message is shown.
  • Errors are shown to the user and logged for auditing.
  • The family name cannot be left blank.

Recommendations

Use names that reflect the area or role function to make them easy to identify.
Review the patent list before confirming to avoid granting unnecessary permissions.
