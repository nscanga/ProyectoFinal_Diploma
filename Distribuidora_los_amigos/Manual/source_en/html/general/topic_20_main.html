System Main Screen

Introduction

The main screen is the control center of the Distribuidora Los Amigos Management System. From here you can access all system functionalities according to the permissions assigned to your user.

The system is designed with an MDI (Multiple Document Interface) that allows working with multiple windows simultaneously.

Main Screen Components

1. Menu Bar

Contains all system modules organized by categories:

  • File: System exit options
  • Management: Access to users, roles and permissions
  • Customers: Customer administration
  • Products: Product catalog and management
  • Suppliers: Supplier management
  • Stock: Inventory control
  • Orders: Order management
  • Reports: Reports and statistics
  • Backup: Backups and restoration
  • Language: System language change

2. Status Bar

Located at the bottom, displays:

  • Currently logged in user
  • Current system date and time
  • Database connection status

3. MDI Work Area

Central space where different module windows open. Can have multiple windows open simultaneously.

First System Access

1 Login

When launching the application, the login window will appear. Enter your username and password provided by the administrator.

2 Permission Verification

The system will automatically load your permissions and enable only the menu options you have access to.

3 Language Configuration

The system will remember the last selected language. You can change it anytime from the Language menu.

System Navigation

Opening a Module

  1. Click on the corresponding menu (e.g. Customers)
  2. Select the desired option (e.g. Create Customer)
  3. A new window will open in the work area

Working with Multiple Windows

  • You can have several windows open at the same time
  • Use the Windows menu to organize open windows
  • Click on a window to bring it to front
Use shortcut keys Alt + [Underlined Letter] to quickly access menu options.

Getting Contextual Help

In any system window you can press F1 to get specific help about that functionality.

ℹ Note: Help will be displayed in the currently selected system language.

Logout

To safely exit the system:

  1. Go to File menu
  2. Click on Exit
  3. Confirm closure if there are open windows
⚠ Important: Always close the system properly to avoid losing unsaved data.

System Configuration

Change Language

The system supports multiple languages:

  • Spanish (es-ES)
  • English (en-US)
  • Portuguese (pt-PT)

Language change: Language Menu → Select language → System updates automatically

Customization

According to your permissions, you can customize:

  • Interface language
  • Window layout

Security Aspects

Aspect Description
Single Session A user can only have one active session at a time
Audit All actions are recorded in the event log
Granular Permissions Each action requires a specific permission
Data Integrity DVH and DVV system to detect tampering

Keyboard Shortcuts

Key Function
F1 Contextual help
Alt + F4 Close current window
Ctrl + S Save (in edit forms)
Esc Cancel current operation
