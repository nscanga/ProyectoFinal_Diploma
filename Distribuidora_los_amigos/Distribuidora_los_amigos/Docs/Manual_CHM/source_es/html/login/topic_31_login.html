Iniciar Sesión en el Sistema

Introducción

El inicio de sesión es el primer paso para acceder al Sistema de Gestión Distribuidora Los Amigos. Este proceso verifica su identidad y carga los permisos asociados a su usuario.

?? Importante: Cada usuario tiene credenciales únicas que no deben ser compartidas con otras personas. El sistema registra todas las acciones asociadas a su usuario.

Cómo Iniciar Sesión

1 Ejecutar la Aplicación

Abra la aplicación "Distribuidora Los Amigos" desde el acceso directo del escritorio o desde el menú de inicio de Windows.

2 Pantalla de Login

Al iniciar la aplicación, aparecerá automáticamente la ventana de inicio de sesión.

3 Ingresar Credenciales

Complete los siguientes campos:

  • Usuario: Su nombre de usuario (sin espacios, sensible a mayúsculas/minúsculas)
  • Contraseña: Su contraseña personal (oculta por seguridad)
4 Seleccionar Idioma (Opcional)

Puede seleccionar el idioma de la interfaz antes de iniciar sesión. Opciones disponibles:

  • ???? Español
  • ???? Inglés
  • ???? Portugués
5 Iniciar Sesión

Click en el botón "Iniciar Sesión" o presione Enter.

Descripción de Campos

Campo Descripción Formato Obligatorio
Usuario Nombre de usuario asignado por el administrador Texto alfanumérico (sensible a mayúsculas) Sí
Contraseña Contraseña personal del usuario Texto (caracteres ocultos por seguridad) Sí
Idioma Idioma de la interfaz del sistema Lista desplegable No (predeterminado: último usado)

Validaciones del Sistema

Durante el inicio de sesión, el sistema verifica:

  • ? Que el usuario exista en el sistema
  • ? Que la contraseña sea correcta
  • ? Que el usuario esté activo (no bloqueado)
  • ? Que no haya otra sesión abierta con el mismo usuario
  • ? La integridad de los datos del usuario (DVH)

Primer Acceso al Sistema

?? Cambio de Contraseña Obligatorio: Si es su primer acceso o el administrador restableció su contraseña, el sistema le solicitará cambiarla inmediatamente por razones de seguridad.

Requisitos para la Nueva Contraseña

  • Mínimo 8 caracteres
  • Al menos una letra mayúscula
  • Al menos una letra minúscula
  • Al menos un número
  • Al menos un carácter especial (!@#$%^&*)
  • No puede ser igual a las últimas 3 contraseñas

Errores Comunes y Soluciones

Error: "Usuario o contraseña incorrectos"

Causa: Las credenciales ingresadas no son válidas.
? Soluciones:
  1. Verifique que el nombre de usuario esté escrito correctamente (sensible a mayúsculas)
  2. Asegúrese de que el Bloq Mayús esté desactivado
  3. Verifique que la contraseña sea correcta
  4. Si olvidó su contraseña, use la opción "¿Olvidó su contraseña?"

Error: "El usuario está bloqueado"

Causa: El usuario fue bloqueado por múltiples intentos fallidos o por el administrador.
? Solución:

Contacte al administrador del sistema para desbloquear su cuenta.

Error: "Ya existe una sesión activa para este usuario"

Causa: El sistema detectó que ya hay una sesión abierta con sus credenciales.
? Soluciones:
  1. Verifique si tiene el sistema abierto en otra computadora
  2. Si la aplicación se cerró inesperadamente, espere 5 minutos e intente nuevamente
  3. Contacte al administrador para cerrar la sesión anterior

Error: "Error de conexión con la base de datos"

Causa: El sistema no puede conectarse al servidor de base de datos.
? Soluciones:
  1. Verifique su conexión de red
  2. Verifique que el servidor de base de datos esté en línea
  3. Contacte al administrador de sistemas

Error: "Se detectó una manipulación de datos"

Causa: El sistema detectó inconsistencias en los datos (fallo de DVH).
? Solución:

URGENTE: Contacte inmediatamente al administrador. Este error indica una posible corrupción o manipulación de datos.

Recomendaciones de Seguridad

Consejos para proteger su cuenta:
  • ?? Nunca comparta su contraseña con nadie
  • ?? Cambie su contraseña periódicamente (cada 90 días recomendado)
  • ?? Use contraseñas únicas - no reutilice contraseñas de otros sistemas
  • ?? No anote su contraseña en lugares visibles
  • ?? Cierre sesión cuando no esté usando el sistema
  • ?? No deje el sistema desatendido con su sesión abierta
  • ?? Bloquee su computadora si se aleja de su escritorio
?? Política de Bloqueo: Después de 5 intentos fallidos de inicio de sesión, su cuenta será bloqueada automáticamente por 30 minutos como medida de seguridad.

Recuperación de Contraseña

Si olvidó su contraseña:

  1. Click en el enlace "¿Olvidó su contraseña?" en la pantalla de login
  2. Ingrese su nombre de usuario y correo electrónico registrado
  3. Responda las preguntas de seguridad configuradas
  4. Recibirá un código de recuperación por correo electrónico
  5. Ingrese el código y establezca una nueva contraseña
?? Nota: El código de recuperación tiene una validez de 15 minutos. Si caduca, deberá solicitar uno nuevo.

Después de Iniciar Sesión

Una vez que inicie sesión exitosamente:

  • Se abrirá la Pantalla Principal del sistema
  • El sistema cargará su idioma preferido
  • Se habilitarán solo las opciones a las que tiene permiso
  • Su nombre de usuario aparecerá en la barra de estado
  • Se registrará el inicio de sesión en la bitácora del sistema
