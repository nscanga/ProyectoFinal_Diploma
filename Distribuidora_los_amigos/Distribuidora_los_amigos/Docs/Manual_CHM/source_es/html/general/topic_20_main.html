Pantalla Principal del Sistema

Introducción

La pantalla principal es el centro de control del Sistema de Gestión Distribuidora Los Amigos. Desde aquí podrá acceder a todas las funcionalidades del sistema según los permisos asignados a su usuario.

El sistema está diseñado con una interfaz MDI (Multiple Document Interface) que permite trabajar con múltiples ventanas simultáneamente.

Componentes de la Pantalla Principal

1. Barra de Menú

Contiene todos los módulos del sistema organizados por categorías:

  • Archivo: Opciones de salida del sistema
  • Gestión: Acceso a usuarios, roles y permisos
  • Clientes: Administración de clientes
  • Productos: Catálogo y gestión de productos
  • Proveedores: Gestión de proveedores
  • Stock: Control de inventario
  • Pedidos: Gestión de pedidos
  • Reportes: Informes y estadísticas
  • Backup: Copias de seguridad y restauración
  • Idioma: Cambio de idioma del sistema

2. Barra de Estado

Ubicada en la parte inferior, muestra:

  • Usuario actualmente conectado
  • Fecha y hora actual del sistema
  • Estado de conexión a la base de datos

3. Área de Trabajo MDI

Espacio central donde se abren las ventanas de los diferentes módulos. Puede tener múltiples ventanas abiertas simultáneamente.

Primer Acceso al Sistema

1 Iniciar Sesión

Al ejecutar la aplicación, se mostrará la ventana de login. Ingrese su usuario y contraseña proporcionados por el administrador.

2 Verificación de Permisos

El sistema cargará automáticamente sus permisos y habilitará solo las opciones del menú a las que tiene acceso.

3 Configuración de Idioma

El sistema recordará el último idioma seleccionado. Puede cambiarlo en cualquier momento desde el menú Idioma.

Navegación por el Sistema

Abrir un Módulo

  1. Click en el menú correspondiente (ej. Clientes)
  2. Seleccionar la opción deseada (ej. Crear Cliente)
  3. Se abrirá una nueva ventana en el área de trabajo

Trabajar con Múltiples Ventanas

  • Puede tener varias ventanas abiertas al mismo tiempo
  • Use el menú Ventanas para organizar las ventanas abiertas
  • Click en la ventana para traerla al frente
Use las teclas rápidas Alt + [Letra subrayada] para acceder rápidamente a las opciones del menú.

Obtener Ayuda Contextual

En cualquier ventana del sistema puede presionar F1 para obtener ayuda específica sobre esa funcionalidad.

?? Nota: La ayuda se mostrará en el idioma actualmente seleccionado en el sistema.

Cerrar Sesión

Para salir del sistema de forma segura:

  1. Ir al menú Archivo
  2. Click en Salir
  3. Confirmar el cierre si hay ventanas abiertas
?? Importante: Siempre cierre el sistema correctamente para evitar pérdida de datos no guardados.

Configuración del Sistema

Cambiar Idioma

El sistema soporta múltiples idiomas:

  • Español (es-ES)
  • Inglés (en-US)
  • Portugués (pt-PT)

Cambio de idioma: Menú Idioma ? Seleccionar idioma ? El sistema se actualiza automáticamente

Personalización

Según sus permisos, puede personalizar:

  • Idioma de la interfaz
  • Disposición de ventanas

Aspectos de Seguridad

Aspecto Descripción
Sesión Única Un usuario solo puede tener una sesión activa a la vez
Auditoría Todas las acciones quedan registradas en la bitácora
Permisos Granulares Cada acción requiere un permiso específico
Integridad de Datos Sistema de DVH y DVV para detectar manipulaciones

Atajos de Teclado

Tecla Función
F1 Ayuda contextual
Alt + F4 Cerrar ventana actual
Ctrl + S Guardar (en formularios de edición)
Esc Cancelar operación actual
